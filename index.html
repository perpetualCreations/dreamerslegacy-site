Raspbot

Small robotic vehicle based on Raspberry Pi and Arduino, with carrying capacity for expansion and modification.

Both hardware and software development are active, with production of Raspbot RCA and specialized hardware modules.

Released CAD assets are outdated, revision 2 will be released later once fully tested.

Project software is available on Github, please see documentation for further hardware specification and additional details. Access these pages below.

ABL

Full Name: AnalyticalBlackLight

Collects Google search interest on defined keywords using pytrends, and exports either a CSV or graph image file.

Originally made humourously to try to track students looking up test answers during virtual testing, with questionable results.

Run using command line or through the GUI wrapper. Project software is available on Github, please see documentation for more details on usage.

SWBS

Full Name: Socket Wrapper for Byte Strings

Python3 package serving as a wrapper for the socket library.

Built to simplify projects that use sockets for sending strings and relatively small amounts of data. Reduces logic and prerequisites to a few straightforward functions.

Derived from the Raspbot RCA's comms module, with modifcations.

Project software is available on Github or PyPI, please see documentation for more details on usage. Access these pages below.

SCHEMA

Tool for automated conversion of Markdown documentation into minimalist HTML making use of the SCHEMA stylesheet.

Code quotes available, with color-coding for Python and terminal commands. Additionally comes with (extremely loose) support for ForTheBadge and Shield.io badges. See documentation for additional options.

Run using command line or through the GUI wrapper. Project software is available on GIthub, please see documentation for more details on usage.
